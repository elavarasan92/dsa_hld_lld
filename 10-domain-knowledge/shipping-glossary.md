# 🚢 Hapag-Lloyd — Domain Glossary

## About Hapag-Lloyd
- **Founded**: 1970 (merger of HAPAG 1847 + Norddeutscher Lloyd 1857).
- **HQ**: Hamburg, Germany.
- **Position**: 5th largest container shipping line globally.
- **Fleet**: ~280+ vessels, ~12M TEU annual capacity, ~3M TEU container fleet.
- **Routes**: ~120 services across all continents.
- **Tech hubs**: Hamburg, Chennai (yours), Gdańsk.

## Strategy Pillars (Strategy 2030)
1. Pure Play in Container Shipping & Connecting Hinterland.
2. Sustainability (decarbonization, dual-fuel LNG vessels).
3. Quality leadership.
4. Digital transformation (the reason FIS3 exists).

## Alliances
- **THE Alliance** (2017–Jan 2025): Hapag-Lloyd + ONE + Yang Ming + HMM.
- **Gemini Cooperation** (Feb 2025–): **Hapag-Lloyd + Maersk** — hub-and-spoke network promising 90% schedule reliability.

## Departments
- **FIS3** = Freight Information Systems (where this role sits). Owns booking, schedule, document, pricing platforms.

## Key Products (public-facing)
- **Quick Quotes** — instant rate quotation.
- **Online Business Suite** — booking, BL approval, invoicing.
- **Live Position** — real-time vessel & container tracking.
- **Navigator** — mobile app for customers.

## Container Shipping Glossary
| Term | Meaning |
|---|---|
| **TEU** | Twenty-foot Equivalent Unit (standard capacity measure) |
| **FEU** | Forty-foot Equivalent Unit |
| **20'GP / 40'GP / 40'HC / 20'RF** | Container types: General Purpose, High Cube, Reefer |
| **B/L (BoL)** | Bill of Lading — receipt + contract + title document |
| **HBL / MBL** | House BL (forwarder) / Master BL (carrier) |
| **SI** | Shipping Instructions (customer → carrier) |
| **VGM** | Verified Gross Mass (mandatory weight declaration, SOLAS) |
| **EDI / EDIFACT** | Electronic Data Interchange (industry messaging) |
| **IFTMIN / IFTSAI / COPRAR** | EDIFACT messages: booking, schedule, container release |
| **Demurrage** | Charge: container stays at port too long |
| **Detention** | Charge: container stays outside terminal too long |
| **D&D** | Demurrage & Detention combined |
| **Port Call** | A vessel's visit to a port |
| **POL / POD** | Port of Loading / Port of Discharge |
| **ETD / ETA** | Estimated Time of Departure / Arrival |
| **ATD / ATA** | Actual Time of Departure / Arrival |
| **FCL / LCL** | Full Container Load / Less than Container Load |
| **Free Time** | Days before D&D charges start |
| **Reefer** | Refrigerated container |
| **DG** | Dangerous Goods (hazmat) — IMDG code |
| **SOC / COC** | Shipper-Owned Container / Carrier-Owned Container |
| **Transhipment** | Moving cargo from one vessel to another at a hub port |
| **Hub-and-spoke** | Mainline + feeder model (basis of Gemini Cooperation) |
| **Slot** | Allocated container space on a vessel |
| **NVOCC** | Non-Vessel Operating Common Carrier (forwarder) |
| **THC** | Terminal Handling Charge |
| **BAF / CAF** | Bunker / Currency Adjustment Factor (surcharges) |

## Useful "domain-aware" answers
- When asked about scaling, mention vessel schedule peaks, port congestion impacts.
- When asked about consistency, mention BoL is a legal document → strong consistency required.
- When asked about ML, mention ETA prediction, port congestion forecasting.
- When asked about events, mention container status changes (Gate-In, Loaded, Discharged, Gate-Out).
