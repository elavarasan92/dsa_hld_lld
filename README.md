# 🚢 Hapag-Lloyd Principal Engineer — Interview Prep

Comprehensive prep guide for the **Principal Software Engineer (Java) – Level 5B** role at **Hapag-Lloyd Technology Centre, Chennai**.

> **Stack**: Java/JEE • JPA • Spring Boot • Kafka • PostgreSQL • AWS • Docker/Kubernetes • Jenkins

---

## 📋 Interview Rounds

| # | Round | Focus | Duration |
|---|---|---|---|
| 1 | Online Assessment | DSA + Java MCQs | 60–90 min |
| 2 | Technical 1 | Core Java, JPA, Spring Boot, REST, SQL | 60 min |
| 3 | Technical 2 | Microservices, Kafka, AWS, DevOps | 60 min |
| 4 | System Design | HLD + LLD (Principal-critical) | 60–75 min |
| 5 | Managerial / Director IT | Mentorship, agile, Hamburg-Chennai | 45 min |
| 6 | HR | Comp, culture, relocation | 30 min |

- Read **one topic per day** (~2 weeks coverage).
- Practice **system design out loud** with whiteboard/Miro.
- Run the **mock interviews** as timed exercises.
- Prep **2–3 STAR stories** you can flex into any behavioural question.
